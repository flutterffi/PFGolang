# Intermediate

This folder starts turning syntax into small code organization habits.

## Goal

Connect language basics to real programming tasks:

- pointers
- package design
- tests
- JSON
- files
- HTTP handlers

## How To Use It

```bash
go run ./intermediate/13_pointers
go run ./intermediate/14_packages_and_files
go test ./pkg/... ./intermediate/...
```

## Focus Rule

Open both the lesson and the supporting package when a lesson imports code from `pkg/`.
