# Foundations

This folder is your syntax gym.

## Goal

Learn the language pieces that show up everywhere:

- variables
- control flow
- functions
- structs
- interfaces
- slices and maps
- errors
- goroutines and channels

## How To Use It

Run lessons in order:

```bash
go run ./foundations/01_hello_world
go run ./foundations/02_variables_and_constants
```

## Focus Rule

Do not rush this folder. If functions, loops, or structs still feel fuzzy, stay here longer before moving on.
